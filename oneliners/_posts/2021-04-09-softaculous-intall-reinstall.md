## Softaculous Reinstall
```bash
wget -N http://files.softaculous.com/install.sh
chmod 755 install.sh
./install.sh --reinstall
```
## Softaculous Install
```bash
wget -N http://files.softaculous.com/install.sh
chmod 755 install.sh
./install.sh
```
